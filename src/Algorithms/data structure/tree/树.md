## 二叉树的遍历

### 树上BFS
从树根开始，按照层次依次访问结点，就是树上BFS。树层序遍历是指按照从根节点到叶子节点的层次关系，一层一层的横向遍历各个节点。根据 BFS 的定义可以知道，BFS 所得到的遍历顺序就是一种层序遍历。

# 树常见面试题

树常见面试题如下：
1. 二叉树的前序、中序和后序遍历，以及对应的非递归写法
2. 根据二叉树的遍历结构恢复二叉树，如从前序与中序遍历序列构造二叉树 和从中序与后序遍历序列构造二叉树
3. N叉树的层序遍历
4. 树的深度，如最小深度和最大深度
